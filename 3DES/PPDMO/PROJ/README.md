# FIT FUSION

## Aplicativo de bem estar e saúde:

### Funcionalidades
- Tela Login
- Tela Calcular IMC
- Tela De Treino
- Tela Plano Alimentar
### Sobre
- É um aplicativo que oferece aos usuários uma interface simples e objetiva com um plano alimentar e uma rotina de treino para iniciantes na academia.
